# Baby Dev App — Panduan Deploy ke Vercel

## Cara Upload ke GitHub (PENTING: Jangan upload zip!)

1. Buka https://github.com dan login
2. Klik **New repository** → nama: `baby-app` → Create
3. Di halaman repo kosong, klik **"uploading an existing file"**
4. **Extract zip dulu ke folder**, lalu pilih **SEMUA FILE di dalam folder** (bukan folder-nya)
   - vite.config.js ✓
   - index.html ✓
   - package.json ✓
   - vercel.json ✓
   - Folder src/ ✓
   - Folder public/ ✓
5. Klik **Commit changes**

## Deploy ke Vercel

1. Buka https://vercel.com → Sign up with GitHub
2. Klik **Add New Project** → Import repo baby-app
3. Framework Preset: Vite (auto-detect)
4. Root Directory: Kosong / titik (.) — JANGAN diisi apapun
5. Klik Deploy — tunggu ~1-2 menit

## Pasang di HP sebagai App

iPhone: Safari → buka URL → Share → Add to Home Screen
Android: Chrome → buka URL → ⋮ → Add to Home Screen
